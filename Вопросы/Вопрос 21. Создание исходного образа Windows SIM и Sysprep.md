Объясните, как создать исходный образ с помощью Windows SIM и Sysprep.

## Ответ
В Windows SIM создают файл ответов, на эталонном ПК устанавливают и настраивают Windows, затем запускают Sysprep /generalize /oobe /shutdown и готовят систему к захвату.

## Ключевые термины
- Windows SIM (System Image Manager)
- Sysprep
- Файл ответов
- Эталонный ПК
- Установка Windows
- Настройка Windows
- generalize
- oobe
- shutdown
- Захват образа
