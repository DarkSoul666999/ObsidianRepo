Объясните, каково общие представление о службе удаленного рабочего стола.

## Ответ
Remote Desktop Services дают пользователям удаленный доступ к рабочему столу или приложениям на сервере. Основные роли: Session Host, Connection Broker, Web Access, Gateway и Licensing.

## Ключевые термины
- Remote Desktop Services (RDS)
- Удаленный доступ
- Рабочий стол
- Приложения на сервере
- RDS роли
- Session Host
- Connection Broker
- Web Access
- Gateway
- Licensing
