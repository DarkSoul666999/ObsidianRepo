Объясните, какие способы реализации централизованного решения по безопасности клиентских ОС существуют.

## Ответ
Централизованную безопасность делают через групповые политики, AD, WSUS/SCCM, BitLocker, EFS, Defender, Firewall, PKI, аудит и мониторинг.

## Ключевые термины
- Групповые политики
- Active Directory
- WSUS (Windows Server Update Services)
- SCCM (System Center Configuration Manager)
- BitLocker
- EFS (Encrypting File System)
- Defender
- Firewall
- PKI (Public Key Infrastructure)
- Аудит
- Мониторинг
