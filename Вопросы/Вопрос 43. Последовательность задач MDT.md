Объясните, как использовать последовательность задач MDT для развертывания клиентских образов.

## Ответ
Task sequence MDT задает порядок развертывания: загрузка WinPE, установка образа, драйверы, приложения, домен, USMT и финальная настройка клиента.

## Ключевые термины
- Task sequence MDT
- WinPE (Windows Preinstallation Environment)
- Установка образа
- Драйверы
- Приложения
- Домен
- USMT (User State Migration Tool)
- Финальная настройка клиента
