Объясните, как устанавливать эталонный компьютер с помощью файла ответов.

## Ответ
Файл ответов unattend.xml передают установщику Windows. Он автоматически задает разделы диска, язык, имя компьютера, ключи, домен и другие параметры.

## Ключевые термины
- Файл ответов
- unattend.xml
- Установщик Windows
- Разделы диска
- Язык
- Имя компьютера
- Ключи активации
- Домен
- Автоматическая установка
