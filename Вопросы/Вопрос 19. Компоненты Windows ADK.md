Объясните, какие компоненты входят в состав Windows ADK.

## Ответ
В Windows ADK входят Windows PE, DISM, Windows SIM, USMT, средства развертывания, VAMT и средства оценки/производительности. Они нужны для создания, настройки и переноса образов.

## Ключевые термины
- Windows ADK (Assessment and Deployment Kit)
- Windows PE (Preinstallation Environment)
- DISM (Deployment Imaging Service and Management)
- Windows SIM (System Image Manager)
- USMT (User State Migration Tool)
- Средства развертывания
- VAMT (Volume Activation Management Tool)
- Средства оценки/производительности
