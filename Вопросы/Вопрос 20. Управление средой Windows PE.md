Объясните, как управлять средой предустановки Windows (Windows PE).

## Ответ
Windows PE создают командой copype, добавляют драйверы и пакеты через DISM, настраивают скрипты запуска и используют как загрузочную среду для установки или обслуживания Windows.

## Ключевые термины
- Windows PE (Preinstallation Environment)
- copype
- Драйверы
- Пакеты
- DISM
- Скрипты запуска
- Загрузочная среда
- Установка Windows
- Обслуживание Windows
