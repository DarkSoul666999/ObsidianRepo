Объясните, как установить MDT 2012 и необходимые компоненты.

## Ответ
Устанавливают Windows ADK с Deployment Tools, Windows PE и USMT, затем MDT 2012 и при необходимости WDS. После установки настраивают Deployment Workbench и права на общий ресурс.

## Ключевые термины
- MDT 2012 (Microsoft Deployment Toolkit 2012)
- Windows ADK (Assessment and Deployment Kit)
- Deployment Tools
- Windows PE (Preinstallation Environment)
- USMT (User State Migration Tool)
- WDS (Windows Deployment Services)
- Deployment Workbench
- Права на общий ресурс
