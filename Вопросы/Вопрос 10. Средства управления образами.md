Объясните, какие средства управления образами (Image Management) существуют.

## Ответ
Для управления образами используют DISM, Windows SIM, Sysprep, MDT, WDS и Configuration Manager. DISM обслуживает образ, MDT и SCCM автоматизируют развертывание.

## Ключевые термины
- DISM (Deployment Imaging Service and Management)
- Windows SIM (System Image Manager)
- Sysprep
- MDT (Microsoft Deployment Toolkit)
- WDS (Windows Deployment Services)
- Configuration Manager
- SCCM (System Center Configuration Manager)
