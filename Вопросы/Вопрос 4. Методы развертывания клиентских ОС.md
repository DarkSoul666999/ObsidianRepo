## Вопрос
Объясните, какие методы развертывания клиентских ОС существуют и какие из них наиболее эффективны в среде организации.

## Ответ
Есть ручная установка, установка с носителя, развертывание образа, WDS, MDT Lite Touch и Configuration Manager Zero Touch. В организации эффективнее автоматизированное развертывание через MDT/WDS или Configuration Manager.

## Ключевые термины
- Ручная установка
- Установка с носителя
- Развертывание образа
- WDS (Windows Deployment Services)
- MDT Lite Touch
- Configuration Manager Zero Touch
- Автоматизация
