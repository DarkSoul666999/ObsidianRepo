Объясните, как планировать и реализовывать инфраструктуру обновлений для поддержки клиентских ОС организации.

## Ответ
Для обновлений выбирают WSUS или Configuration Manager, создают группы тестирования, утверждают обновления поэтапно, планируют перезагрузки и контролируют отчеты о соответствии.

## Ключевые термины
- Инфраструктура обновлений
- WSUS (Windows Server Update Services)
- Configuration Manager
- Группы тестирования
- Утверждение обновлений
- Перезагрузки
- Отчеты о соответствии
