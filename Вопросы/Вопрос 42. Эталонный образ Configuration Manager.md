Объясните, как построить эталонный образ на основе последовательности задач Configuration Manager.

## Ответ
В Configuration Manager создают build-and-capture task sequence, добавляют ОС, драйверы, приложения и обновления, выполняют Sysprep и сохраняют готовый WIM.
