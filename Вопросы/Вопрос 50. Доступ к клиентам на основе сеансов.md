Объясните, как настроить доступ к клиентам на основе сеансов (Session-Based Desktop).

## Ответ
Для session-based desktop разворачивают RD Session Host, создают коллекцию, назначают пользователей и публикуют полный рабочий стол или RemoteApps.

## Ключевые термины
- Session-Based Desktop
- RD Session Host
- Коллекция
- Назначение пользователей
- Полный рабочий стол
- RemoteApps
