Объясните, как реализовать решение для восстановления файлов, защищенных EFS.

## Ответ
Для восстановления EFS заранее назначают Data Recovery Agent и хранят его закрытый ключ в безопасном месте. При потере пользовательского ключа агент расшифровывает файлы.

## Ключевые термины
- EFS (Encrypting File System)
- Data Recovery Agent
- Закрытый ключ
- Восстановление файлов
- Расшифровка
