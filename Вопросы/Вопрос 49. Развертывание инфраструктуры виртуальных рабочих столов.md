Объясните, как настроить развертывание инфраструктуры виртуальных рабочих столов.

## Ответ
Для VDI готовят Hyper-V, шаблон виртуальной машины, RD Virtualization Host, Connection Broker и Web Access. Потом создают пул рабочих столов и назначают пользователей.

## Ключевые термины
- VDI (Virtual Desktop Infrastructure)
- Развертывание VDI
- Hyper-V
- Шаблон виртуальной машины
- RD Virtualization Host
- Connection Broker
- Web Access
- Пул рабочих столов
- Назначение пользователей
