Объясните, как настроить шлюзы RDS.

## Ответ
Для RD Gateway устанавливают роль, привязывают SSL-сертификат, создают CAP/RAP, открывают порт 443 и проверяют подключение через RD Web или RDP-клиент.

## Ключевые термины
- RD Gateway
- Установка роли
- SSL-сертификат
- CAP (Connection Authorization Policy)
- RAP (Resource Authorization Policy)
- Порт 443
- RD Web
- RDP-клиент
