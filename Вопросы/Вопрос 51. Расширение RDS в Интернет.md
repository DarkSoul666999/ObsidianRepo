Объясните, как расширить среду Remote Desktop Services в Интернет.

## Ответ
Чтобы открыть RDS в Интернет, используют RD Gateway и RD Web Access через HTTPS. Нужны сертификат, правила firewall/NAT и политики доступа CAP/RAP.

## Ключевые термины
- RDS в Интернете
- RD Gateway
- RD Web Access
- HTTPS
- Сертификат
- Firewall
- NAT
- Политики доступа
- CAP (Connection Authorization Policy)
- RAP (Resource Authorization Policy)
