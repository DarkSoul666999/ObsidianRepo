Объясните, как подготовить среду Zero Touch Installation.

## Ответ
Для ZTI устанавливают и связывают Configuration Manager и MDT, включают PXE, импортируют образы, драйверы и приложения, создают коллекции и task sequence, затем тестируют пилот.

## Ключевые термины
- ZTI (Zero Touch Installation)
- Configuration Manager
- MDT (Microsoft Deployment Toolkit)
- PXE (Preboot Execution Environment)
- Импорт образов
- Импорт драйверов
- Импорт приложений
- Коллекции
- Task sequence
- Пилотное тестирование
