Объясните, как планировать и реализовывать шифрование с помощью EFS для защиты конфиденциальной информации.

## Ответ
Для EFS нужны NTFS, сертификаты пользователей, резервное копирование ключей и агент восстановления данных. EFS лучше применять для отдельных конфиденциальных файлов, а не для всего диска.

## Ключевые термины
- EFS (Encrypting File System)
- NTFS
- Сертификаты пользователей
- Резервное копирование ключей
- Агент восстановления данных
- Конфиденциальные файлы
