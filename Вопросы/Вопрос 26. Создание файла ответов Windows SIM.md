Объясните, как создать файл ответов с помощью Windows SIM.

## Ответ
В Windows SIM открывают install.wim, создают новый answer file, добавляют компоненты по этапам установки, заполняют параметры, проверяют файл и сохраняют unattend.xml.

## Ключевые термины
- Windows SIM (System Image Manager)
- install.wim
- answer file
- Компоненты установки
- Параметры
- unattend.xml
