Объясните, как интегрировать WDS с MDT 2012 для обеспечения возможностей загрузки PXE.

## Ответ
В WDS включают PXE, добавляют LiteTouch boot images для нужной архитектуры, проверяют DHCP/PXE-настройки и тестируют загрузку клиента по сети.

## Ключевые термины
- Интеграция WDS и MDT 2012
- PXE (Preboot Execution Environment)
- LiteTouch boot images
- Архитектура (x86, x64)
- DHCP-настройки
- PXE-настройки
- Загрузка клиента по сети
